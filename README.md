# TourisMore  
Приложение для прогулок по городу с нестандартными маршрутами

TourisMore - Android-приложение, созданное в рамках летней школы мобильной разработки ВШЭ командой из 4 человек.  
Приложение помогает пользователям находить необычные прогулочные маршруты, созданные экспертами и самими жителями города.

## Основной функционал

###  Маршруты
- нестандартные авторские маршруты  
- фильтрация по тематикам  
- просмотр описания каждой точки  
- фотографии и дополнительные материалы  

###  Аудиогид
- встроенный аудиоплеер  
- поддержка озвученных маршрутов  
- истории и локальные факты от жителей  

###  Карта
- интеграция с **Yandex Maps API**  
- отображение маршрута на карте  
- навигация до следующей точки  

### Авторские маршруты
- создание своих маршрутов (карта + фото + описание)  
- возможность публикации и продажи маршрутов
